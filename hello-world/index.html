Hello World

March 5, 2026

This is the first post on the blog. Written in Obsidian, published with Astro.

Why blog from Obsidian?

I already write notes here every day. Turning some of them into blog posts should be frictionless — same editor, same workflow, just a different folder.

The setup is simple: markdown files in 08 Blog/, built by Astro, deployed to GitHub Pages. No CMS, no database, no JavaScript on the page.

How wikilinks work

Links to other blog posts like Hello World become real links. Links to things outside the blog like React Native render as plain text — they’re useful context while writing but don’t need to be clickable on the published site.
